Select certificate location

You have to select an eIDAS certificate to sign. There are two possibilities for the location of such certificate:

  • The certificate is in the Windows certificate store, a secure store managed by the Windows system.
  • The certificate is in a file in your disk. This mechanism can be used in any system, but is typically used in Linux systems.

Please, select the certificate using the mechanism most appropriate for you.

{{if eq .os "windows"}}
Select from Windows certstore

The application will query the store and present the certificates so you can select the one you want to use.

Click the button below to start the process.

Select from certstore
{{end}}
Select from file

Select the certificate file using the button below. If the file is encrypted (which it should), you have also to provide the password that was used to encrypt the certificate file.
